# 1.0

可以自定义角色图片和声音，赛博逗铜仁女棒，嘿嘿

操作方式：把你自己的图片和音频分别重命名为对应的名称（takina.png-sakana.mp3,chisato.png-chinanako.mp3）替换原有图片就好了。

该项目的开发基于卜卜口大佬制作的Sakana！石蒜模拟器（https://github.com/itorr/sakana）

本人对代码了解程度基本为0，全程使用deepseek修改与制作。
